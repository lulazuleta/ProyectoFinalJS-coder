    Proyecto Final


    Para el proyecto final del curso de JavaScript, decidi hacer un juego, el cual se encuentra situado en Egipto. El objetivo del mismo es poder salir de una piramide, mediante algunos acertijos que se van presentando.
    Como secciones adicionales, hay una galeria de fotos, donde se pueden ver imagenes relacionadas al escenario del juego.
    Una seccion de contacto, donde hay un formulario para poder enviar correos.
    Dentro de este proyecto se utilizan varias herramientas como HTML, CSS, JavaScript, Dom, Canvas, Jquery, etc.

    Espero que este proyecto sea de su agrado.
